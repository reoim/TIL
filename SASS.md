# How to install SASS
**Linux** (need to install Ruby first)
```
sudo su -c "gem install sass"
```
**Mac**
```
gem install sass
```
Check if it is installed properly
```
sass -v
```

# Compass
Compass is CSS authoring framework. 
```
gem install compass
```
> ### Watching and compiling
> Keep CSS file up to date when changes are made
>```
$ cd /path/to/project
$ compass watch
```
